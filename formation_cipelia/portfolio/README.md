# Portfolio Pierre Bourgeois - Consultant Data

## 📋 Description

Portfolio web professionnel de Pierre Bourgeois, Consultant Data Scientist chez AVISIA. Ce site met en avant :
- Formation et parcours éducatif
- Expériences clients et projets réalisés
- Compétences techniques (Data Science, Engineering, DevOps)
- Rôles transversaux (Management, Formation, Coaching)
- Secteurs d'expertise
- Informations de contact

## 🚀 Utilisation

### Ouvrir le site localement
1. Double-cliquez sur `index.html` OU
2. Faites un clic droit sur `index.html` → "Ouvrir avec" → Votre navigateur favori

### Structure des fichiers
```
portfolio/
├── index.html      # Page principale
├── styles.css      # Feuille de styles
├── script.js       # Interactivité JavaScript
└── README.md       # Ce fichier
```

## ✨ Fonctionnalités

### Design
- **Responsive** : Compatible mobile, tablette et desktop
- **Moderne** : Dégradés, ombres, animations fluides
- **Couleurs** : Bleu professionnel (#0066cc) et accent rouge (#ff6b6b)

### Navigation
- Menu de navigation sticky (collant en haut)
- Liens d'ancrage pour sauter rapidement aux sections
- Smooth scroll (défilement fluide)

### Sections
1. **Accueil** : Présentation avec appel à l'action
2. **À propos** : Description + statistiques animées
3. **Formation** : Timeline de la formation
4. **Expériences** : Projets clients détaillés par secteur
5. **Rôles transversaux** : Manager, Formateur, Coach, etc.
6. **Compétences** : Catégorisées par domaine
7. **Points forts** : Résumé des atouts clés
8. **Contact** : Email, téléphone, LinkedIn, localisation

### Animations
- Fade-in progressif du contenu au scroll
- Animations au hover sur les cartes
- Compteurs animés pour les statistiques
- Transitions fluides partout

## 📱 Responsive Design

Le site s'adapte automatiquement à tous les écrans :
- **Desktop** : Largeur maximale 1200px, 2-3 colonnes
- **Tablette** : 1-2 colonnes selon l'appareil
- **Mobile** : 1 colonne, menu optimisé

## 🎯 Personnalisation

### Modifier les couleurs
Dans `styles.css`, ligne 1, changez les variables :
```css
:root {
    --primary-color: #0066cc;      /* Bleu principal */
    --secondary-color: #ff6b6b;    /* Rouge accent */
    --dark-color: #1a1a1a;         /* Noir foncé */
    --light-color: #f8f9fa;        /* Gris clair */
}
```

### Modifier le contenu
- Ouvrez `index.html` dans un éditeur de texte
- Cherchez la section à modifier
- Changez le texte, les dates, les informations
- Sauvegardez et rafraîchissez le navigateur

### Ajouter une nouvelle section
1. Copiez une section existante
2. Changez l'ID et le contenu
3. Ajoutez un lien dans le menu navbar
4. Ajoutez le CSS correspondant si nécessaire
