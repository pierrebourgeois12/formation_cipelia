🎯 3 CONCEPTS CLÉS
═══════════════════════════════════════════════════════════════════════════════

1️⃣  TOOLS (Outils) - Ce que Claude peut APPELER
   ────────────────────────────────────────────
   
   Dans src/main.py:
   
   @mcp.tool()
   async def hello(name: str) -> str:
       """Salue quelqu'un"""
       return f"Bonjour {name}!"
   
   Claude peut appeler: hello(name="Pierre")


2️⃣  RESOURCES (Ressources) - Les DONNÉES de Claude
   ──────────────────────────────────────────────────
   
   Fichiers dans resources/:
   - example.txt (document)
   - data.json (données structurées)
   
   Claude peut les lire via read_resource(filename)


3️⃣  PROMPTS (Instructions) - GUIDER Claude
   ──────────────────────────────────────────
   
   Fichiers dans prompts/:
   - expert_math.md
   - formation_llm.md
   - assistant_data.md
   
   Copiez le contenu au début de votre conversation

🔧 OUTILS DISPONIBLES
═══════════════════════════════════════════════════════════════════════════════

  → Salue quelqu'un
  → Exemple: hello(name="Pierre")

calculate(operation: str, a: float, b: float)
  → Calcule: "add", "subtract", "multiply", "divide"
  → Exemple: calculate(operation="add", a=10, b=5)

list_resources()
  → Liste les ressources disponibles

read_resource(filename: str)
  → Lit le contenu d'une ressource
