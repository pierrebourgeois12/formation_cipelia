# Exemple de Ressource

C'est un fichier texte dans le dossier `resources/`.

Claude peut accéder à ce fichier en utilisant l'outil `read_resource("example.txt")`.

## Comment ça marche?

1. Vous créez des fichiers ici (JSON, TXT, CSV, etc.)
2. Le serveur MCP les rend accessibles à Claude
3. Claude peut appeler l'outil `read_resource()` pour les lire
4. Cela peut servir pour:
   - Des documents (PDFs convertis en texte)
   - Des données (JSON, CSV)
   - Des instructions (prompts, guides)
   - Des exemples

## Fichiers disponibles

Utilisez l'outil `list_resources()` pour voir tous les fichiers.
